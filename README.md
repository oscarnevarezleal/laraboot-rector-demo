# laraboot-rector-demo

This project shows how to generate a Laravel application using Laraboot CLI.
Additionaly it runs rector using custom configuration taken from the `rector.php` file.

[![Open in Gitpod](https://gitpod.io/button/open-in-gitpod.svg)](https://www.gitpod.io/#https://github.com/oscarnevarezleal/laraboot-rector-demo/tree/main)

## Features

- [x] Use PHP 8
- [x] Add **Rector** support using `@core/laraboot-rector` [build task](https://laraboot.io/buildtask/)
- [x] Gitpod integration
